# modern-warfare-2-cfg-creator
An all-in-one creator suite for editors to easily adjust the environment of mw2 (2009)

Download "download iw4xstroke" file 
Open mw2 (2009)
\devmap mp_(mapname)
type \seta r_filmUseTweaks 1 & \seta r_filmTweakEnable 1
Open tool as admin
